# Support

## Before you ask

1. Read **QUICKSTART.md** — it covers install, first run, and the no-paid-API path.
2. Check the **compatibility matrix** in QUICKSTART (which skills need which
   optional tools, and what each does without them).
3. Confirm you're on a supported Claude Code version and that the plugin loaded
   (`/plugin` should list `designer-pro-and-seo`).

## Getting help

- **Channel:** open an issue on GitHub —
  <https://github.com/ZachArticulateV/designer-pro-and-seo/issues>.
- **Include:** what you ran, the skill name, what you expected, what happened,
  and your OS + Claude Code version. Redact any API keys or client data.

## Scope of support

- **Covered:** install problems, a skill not firing or erroring, a script
  failing, unclear documentation, and bug reports.
- **Not covered:** writing your deliverables for you, configuring paid
  third-party APIs you obtained elsewhere, or issues caused by modifying the
  plugin's files.

## Status & expectations

This is a **free, open-source plugin** in active development. The shipping skill
set (see README) is the supported surface; skills marked *In development* are
works in progress and excluded from support until released.

Help is best-effort by the maintainer and the community via GitHub — there is no
service-level agreement.
